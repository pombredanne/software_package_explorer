Software Package Explorer

On a Debian and Ubuntu systems, there is a file called /var/lib/dpkg/status that holds information about software packages that the system knows about. This web application exposes some key information about packages in the aformentioned file by providing you an easy-to-use interface to navigate, explore and learn about such packages. Begin your journey by navigating to the home page by clicking the link Packages in the top-left-hand-corner and deep dive straight into the core of this system's software packages and their dependencies by exploring each package link.

This web application is built using Python3.6 and Flask.
